# Research Inspirations

Sources that shaped Quantum Lens design. Each contributed specific implementation decisions.

## Source Registry

| Source | Key Takeaway | What We Stole |
|--------|-------------|---------------|
| **PRISM Framework** (Diamond 2025) | 7 neuroscience-grounded worldviews with Pareto conflict-mediation | Basis perspectives grounded in cognitive modes, not arbitrary categories. Conflict surfacing over conflict resolution. |
| **Multi-LLM Debate** (NeurIPS 2024) | Same-model debate converges to shared misconceptions | Anti-convergence rules: lenses MUST be architecturally different, not just "different prompts" |
| **Quantum Abduction** (Pareschi 2025) | Hypotheses in superposition, interference before elimination | Synthesis happens BEFORE pruning. Let contradictions coexist and interfere. |
| **Superpositional Belief** (2026) | Ambiguity is structured and actionable, deferred commitment | Don't force collapse. Present superposition AS output. Goal-conditioned collapse. |
| **Neurodiverse AI** (Vallverdu 2025) | Autistic systemizing, ADHD divergent linking, dyslexic spatial reasoning | Cognitive mode lenses modeled on computationally distinct strategies |
| **Team of Rivals** (Vijayaraghavan 2025) | Coherence from structured disagreement | Perspectives designed to DISAGREE by construction |
| **Adversarial Collaboration** (FRI 2024) | Crux identification: minimal factual claim resolving most disagreement | When lenses disagree, auto-identify the CRUX - turns vague disagreement into investigation targets |
| **Structure-Mapping Engine** (Gentner 1986+) | Analogy preserves relational structure, not surface features | Cross-perspective isomorphism detection: find where different lenses see the same deep pattern |
| **Atoms of Thought** (Teng 2025) | Decompose into irreducible atoms with Markov property | AoT decomposition BEFORE lens analysis. Per-atom interference detection. |
| **Atom-Searcher** (Deng 2025) | Semantic tagging: [observation], [inference], [assumption], [prediction] | Tag lens outputs for precise interference detection |
